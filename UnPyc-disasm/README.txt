--== Installation ==--
No installation is required, but make sure, that you have python and python-tk (only for Gui mode) installed on your box.
Functionallity was tested on Python 2.5.2.

--== Help ==--
Usage: ./UnPyc [dvvDgcVh] <file>
d  - disassemble
x  - show xref`s (used with d)
v  - verbose (used with d)
vv - very verbose (used with d)
D  - decompile (not implemented yet)
g  - gui (control flows)
c  - copyright
V  - version
h  - help
